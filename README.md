
# YouTube Clone

#Overview
A responsive YouTube Clone web application built using HTML, CSS, and JavaScript. The project replicates the core look and functionality of the popular video sharing platform, enabling users to browse, search, and watch videos in a sleek, user-friendly interface.

Features
	•	Browse and play embedded videos through a custom video player
	•	Video details view (title, description, uploader, views, likes, etc.)
	•	Responsive design for both desktop and mobile devices
	•	Search bar for quick video queries
	•	Interactive UI elements (sidebar, menus, etc.)
	•	(Optional) Comments, likes, and subscription buttons
	•	(Optional) User authentication and video upload support
  
Technologies Used
	•	HTML – Structure and layout
	•	CSS – Styling and responsive design
	•	JavaScript – Functionality and interactivity
  
Getting Started
To run this project locally:
	1.	Clone the repository or download the source files.
	2.	Open the  index.html  file in any modern web browser.
	3.	Browse and play videos, try out searching and navigating through UI components.
